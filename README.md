# 京东 Cookie 提取器

一个 Chrome 浏览器扩展，用于从手机版京东 (m.jd.com) 提取登录凭证 `pt_key` 和 `pt_pin`。

## 功能

- 点击按钮直接打开京东手机版
- 一键提取 Cookie 中的 `pt_key` 和 `pt_pin`
- 自动复制到剪贴板
- 符合京东品牌风格的图标

## 安装

1. 下载或克隆本仓库
2. 打开 Chrome → `chrome://extensions/`
3. 开启右上角 **"开发者模式"**
4. 点击 **"加载已解压的扩展程序"**
5. 选择本仓库文件夹

## 使用方法

1. 点击插件图标，点击 **"打开京东"**
2. 在打开的页面中登录你的京东账号
3. 再次点击插件图标，点击 **"提取Cookie"**
4. 文本框会显示格式化的 Cookie：`pt_key=xxx;pt_pin=xxx;`
5. 点击 **"复制"** 按钮复制到剪贴板
